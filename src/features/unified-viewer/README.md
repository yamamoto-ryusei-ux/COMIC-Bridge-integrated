# unified-viewer

統合ビューアー。5スロットパネル（左端/左サブ/中央ビューアー/右サブ/右端）に任意タブを配置。共通タブ: ファイル/レイヤー/写植仕様/テキスト/校正JSON/テキスト照合。PDF/PSD/画像の高解像度表示、単ページ化、ページ連動、タブ入れ替え記憶（displacedTabs）。

**関連**: `unified-viewer/`（UnifiedViewer, UnifiedSubComponents, utils.ts, useViewerFileOps, ProgenImageViewer）、`views/UnifiedViewerView.tsx`, `views/ViewerView.tsx`、`store/unifiedViewerStore.ts`、`hooks/useHighResPreview.ts`, `useOpenInPhotoshop.ts`、`kenban-utils/textExtract.ts`（LCS diff、テキスト照合用）。
