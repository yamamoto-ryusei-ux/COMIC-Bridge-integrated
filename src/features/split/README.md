# split

**関連**: `components/split/`（SplitPanel, SplitPreview, SplitResultDialog）、`views/SplitView.tsx`、`hooks/useSplitProcessor.ts`、`store/splitStore.ts`、`src-tauri/scripts/split_psd.jsx`。
