# layer-control

レイヤー制御（Photoshop JSX経由）。5アクションモード: hide/show/custom/organize/layerMove。サブタブ構成（レイヤー制御/リネーム）、保存先選択（上書き/別フォルダ）、詳細レポートダイアログ、LayerPreviewPanel（レイヤー構造/ビューアータブ切替）。

**関連**: `components/layer-control/`（LayerControlPanel, LayerPreviewPanel, LayerControlResultDialog）、`views/LayerControlView.tsx`、`hooks/useLayerControl.ts`、`store/layerStore.ts`、`src-tauri/scripts/hide_layers.jsx`, `organize_layers.jsx`, `move_layers.jsx`, `custom_operations.jsx`, `lock_layers.jsx`, `merge_layers.jsx`。
